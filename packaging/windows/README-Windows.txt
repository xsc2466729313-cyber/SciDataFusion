SciDataFusion Windows x64 便携版
================================

运行方法
--------
1. 完整解压本 ZIP，不要直接在压缩包内运行。
2. 双击 SciDataFusion.exe 或 Start-SciDataFusion.bat。
3. 保持启动窗口开启，浏览器会自动打开中文工作台。
4. 关闭启动窗口即可停止本地服务。

联网配置
--------
首次启动默认使用离线演示，不需要 API Key。
需要联网搜索时，在页面左侧打开“联网配置”，填写：
- SerpApi Key
- 阿里云百炼 API Key
- 百炼 Base URL（页面已提供默认北京地址）

配置仅保存到当前解压目录的 .env 文件。请勿分享该文件。

系统要求
--------
- Windows 10/11 64 位
- 无需安装 Python、uv 或 Git
- 联网模式需要能够访问 SerpApi 和阿里云百炼
