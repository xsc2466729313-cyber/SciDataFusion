# M29 验收: 中文工作台可靠性与证据关系

- [ ] 联网配置未完成时，前端不提交任务并打开配置页。
- [ ] 本机工作台可以保存配置，刷新和容器重启后仍显示配置状态。
- [ ] 远程地址即使伪造代理标记也不能写配置。
- [ ] 任务异常保存稳定错误码、中文说明和恢复动作，响应不包含内部异常详情。
- [ ] 服务端日志保留任务编号和堆栈，方便定位中途失败。
- [ ] 任务状态、图谱类型、图谱关系、配置说明和操作按钮均使用中文。
- [ ] 在线结构化数据至少包含数据集到证据、证据到字段、证据间共现关系。
- [ ] 共现关系只表示同一原始记录，不声称因果、一致或等价。
- [ ] Ruff、mypy、pytest、Bandit、秘密扫描和前端生产构建全部通过。
- [ ] Docker 运行态完成配置保存、失败预检、中文图谱和离线任务回归验收。
